# Student Management System

## 📌 Project Description
This is a Java-based Student Management System that allows users to manage student records using file handling.

## 🚀 Features
- Add Student
- View Students
- Save student data to file
- Simple service-based architecture

## 🛠 Technologies Used
- Java
- OOP Concepts
- File Handling

## ▶ How to Run

1. Compile the program:
   javac Main.java

2. Run the program:
   java Main
